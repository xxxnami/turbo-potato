How to use | 使用方法

1. Install File Converter.
   安装 File Converter。
2. Unzip Languages.zip.
   解压 Languages.zip。
3. Use the extracted Languages folder to replace the Languages folder in the File Converter installation folder.
   将解压出来的 Languages 文件夹用于替换 File Converter 安装文件夹中的 Languages 文件夹。
